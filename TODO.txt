Lab 2 Assignment - New CreateFighterController Implementation with Form Validation

Overview
Implement a CreateFighterController with POST method to process fighter registration and add it to the fighter pool.
Fighter pool is a list of fighters similar to CreatePlayerController.java
 
Requirements

Controller Implementation
[X]	Implement `createFighter()` method with POST signature

[X]	All fields required: name, health, damage, resistance

[X]	Damage validation: Must be less than 100
